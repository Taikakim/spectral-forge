# UI Parameter Specification — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the UI parameter contract from the spec: a centralized `CurveDisplayConfig` table, a new per-curve curvature transform (smoothstep tilt shaping), a shared hover text routine, and UI scaling helpers — so that display parameters never drift across code changes.

**Architecture:** A new `src/editor/curve_config.rs` is the single source of truth for all per-curve display properties. The curvature control is a new `FloatParam` generated by `build.rs` alongside the existing tilt/offset params; it is applied in both the DSP pipeline and the GUI display path. UI scaling helpers live in `theme.rs`. All participating functions carry a comment pointing at the spec.

**Tech Stack:** Rust, nih-plug, egui. No new crate dependencies.

**Spec:** `docs/superpowers/specs/2026-04-23-ui-parameter-spec-design.md`

Replace the entire function with:

```rust
/// Apply spectral tilt (pivoted at 1 kHz), additive offset, and curvature (S-curve bend)
/// to a slice of per-bin curve gains, then clamp to [0, ∞).
/// See docs/superpowers/specs/2026-04-23-ui-parameter-spec-design.md §2.
pub fn apply_curve_transform(
    gains: &mut [f32],
    tilt: f32,
    offset: f32,
    curvature: f32,
    sample_rate: f32,
    fft_size: usize,
) {
    if gains.is_empty() { return; }
    if tilt.abs() < 1e-6 && offset.abs() < 1e-6 { return; }
    const LOG_20: f32 = 1.301_030;
    const LOG_RANGE: f32 = 3.0;
    const PIVOT: f32 = 0.566_32; // log10(1000/20) / log10(20000/20)
    // Smoothstep value at the pivot — used to zero the sigmoid shape at the pivot.
    const S_PIVOT: f32 = 3.0 * PIVOT * PIVOT - 2.0 * PIVOT * PIVOT * PIVOT;
    for (k, g) in gains.iter_mut().enumerate() {
        let freq_hz = (k as f32 * sample_rate / fft_size as f32).max(20.0);
        let norm = ((freq_hz.log10() - LOG_20) / LOG_RANGE).clamp(0.0, 1.0);
        let linear_shape  = norm - PIVOT;
        let s             = 3.0 * norm * norm - 2.0 * norm * norm * norm;
        let sigmoid_shape = s - S_PIVOT;
        let shape = linear_shape + curvature * (sigmoid_shape - linear_shape);
        let t = tilt * shape;
        *g = ((*g + offset) * (1.0 + t)).max(0.0);
    }
}
```

```rust
/// `curvature` ∈ [0, 1]: 0 = straight tilt, 1 = full smoothstep S-curve pivoted at 1 kHz.
/// See docs/superpowers/specs/2026-04-23-ui-parameter-spec-design.md §2.

    freq_hz: f32,
    tilt: f32,
    offset: f32,

    const LOG_RANGE: f32 = 3.0;
    const S_PIVOT: f32 = 3.0 * TILT_PIVOT_NORM * TILT_PIVOT_NORM
                       - 2.0 * TILT_PIVOT_NORM * TILT_PIVOT_NORM * TILT_PIVOT_NORM;
    let norm = ((freq_hz.max(20.0).log10() - LOG_20) / LOG_RANGE).clamp(0.0, 1.0);
    let linear_shape  = norm - TILT_PIVOT_NORM;
    let s             = 3.0 * norm * norm - 2.0 * norm * norm * norm;
    let sigmoid_shape = s - S_PIVOT;
    let shape = linear_shape + curvature * (sigmoid_shape - linear_shape);
    let t = tilt * shape;
    ((gain + offset) * (1.0 + t)).max(0.0)
}
```
