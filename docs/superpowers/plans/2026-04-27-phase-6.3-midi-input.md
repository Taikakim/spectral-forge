> **STATUS:** PLANNED, NOT STARTED. Authoritative status: `docs/superpowers/STATUS.md`.

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development`

> Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wire nih-plug MIDI input into Spectral Forge so modules can read held notes
(`[bool; 128]`) and held pitch classes (`[bool; 12]`) via `ModuleContext`.

**Architecture:** Set `Plugin::MIDI_INPUT = MidiConfig::Basic` in `lib.rs`. In
`SpectralForge::process()`, drain `NoteEvent::NoteOn / NoteOff` from `ProcessContext`
into `Pipeline::held_notes` and recompute `Pipeline::held_pitch_classes`. Inside the
hop closure, expose `Some(&held_notes)` and `Some(&held_pitch_classes)` via
`ModuleContext`. Keeps the audio thread RT-safe (no allocation, no locking).

**Tech stack:** nih-plug `MidiConfig::Basic`, `NoteEvent` enum, existing
`Pipeline::process()` signature, Phase 1's `ModuleContext` Option fields
`midi_notes: Option<&'block [bool; 128]>` and `held_pitch_classes: Option<&'block [bool; 12]>`.

**Cross-phase notes:** Assumes Phase 1 added the Option fields to `ModuleContext`.
This plan populates them.

// src/dsp/midi.rs

/// Number of distinct MIDI notes (0..=127).
pub const NUM_MIDI_NOTES: usize = 128;
/// Number of pitch classes (C..B).
pub const NUM_PITCH_CLASSES: usize = 12;

/// Mark `note` held; set its pitch-class bit.
/// Notes outside 0..128 are silently ignored.
#[inline]
pub fn apply_note_on(note: u8, held: &mut [bool; NUM_MIDI_NOTES], classes: &mut [bool; NUM_PITCH_CLASSES]) {
    let n = note as usize;
    if n >= NUM_MIDI_NOTES { return; }
    held[n] = true;
    classes[n % NUM_PITCH_CLASSES] = true;
}

/// Release `note`. The pitch-class bit is recomputed; it stays set if any other
/// held note maps to the same class (e.g. C4 release while C5 still held).
/// Notes outside 0..128 are silently ignored.
#[inline]
pub fn apply_note_off(note: u8, held: &mut [bool; NUM_MIDI_NOTES], classes: &mut [bool; NUM_PITCH_CLASSES]) {
    let n = note as usize;
    if n >= NUM_MIDI_NOTES { return; }
    held[n] = false;
    let pc = n % NUM_PITCH_CLASSES;
    let mut still_held = false;
    let mut k = pc;
    while k < NUM_MIDI_NOTES {
        if held[k] { still_held = true; break; }
        k += NUM_PITCH_CLASSES;
    }
    classes[pc] = still_held;
}

/// Reset both arrays to all-false. Called on Pipeline::reset().
#[inline]
pub fn clear_midi_state(held: &mut [bool; NUM_MIDI_NOTES], classes: &mut [bool; NUM_PITCH_CLASSES]) {
    *held    = [false; NUM_MIDI_NOTES];
    *classes = [false; NUM_PITCH_CLASSES];
}
```

Add to `src/dsp/mod.rs`:

```

Rename the unused param and drain events:

```rust
    fn process(
        &mut self,
        buffer: &mut Buffer,
        aux: &mut AuxiliaryBuffers,
        ctx: &mut impl ProcessContext<Self>,
    ) -> ProcessStatus {
        dsp::guard::flush_denormals();
        if let (Some(pipeline), Some(shared)) = (&mut self.pipeline, &mut self.shared) {
            // Drain MIDI events before audio processing — the held-notes state used
            // by the closure must reflect the start-of-block keyboard state.
            // Sample-accurate per-hop note timing is deferred to v2 (see Phase 6 umbrella).
            while let Some(event) = ctx.next_event() {
                use nih_plug::prelude::NoteEvent;
                match event {
                    NoteEvent::NoteOn { note, .. }  => pipeline.note_on(note),
                    NoteEvent::NoteOff { note, .. } => pipeline.note_off(note),
                    _ => {}
                }
            }
            pipeline.process(buffer, aux, shared, &self.params);
        }
        ProcessStatus::Normal
    }
```

use spectral_forge::dsp::pipeline::{Pipeline, FFT_SIZE};

#[test]
fn pipeline_note_on_off_roundtrip() {
    let sr  = 48000.0;
    let fft = FFT_SIZE;
    let mut slot_types = [ModuleType::Empty; 9];
    slot_types[8] = ModuleType::Master;
    let mut p = Pipeline::new(sr, 2, fft, &slot_types);

    assert!(p.held_notes()[60]);
    p.note_on(64);
    assert!(p.held_notes()[64]);
    p.note_off(60);
    assert!(!p.held_notes()[60]);
    assert!(p.held_notes()[64]);
}

#[test]
fn pipeline_reset_clears_midi_state() {
    let sr  = 48000.0;
    let fft = FFT_SIZE;
    let mut slot_types = [ModuleType::Empty; 9];
    slot_types[8] = ModuleType::Master;
    let mut p = Pipeline::new(sr, 2, fft, &slot_types);
    p.note_on(60);
    p.note_on(72);
    assert!(p.held_notes()[60]);
    p.reset(sr, 2);
    assert!(p.held_notes().iter().all(|&n| !n),
