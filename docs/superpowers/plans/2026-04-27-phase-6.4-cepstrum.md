**Goal:** Lazy per-hop cepstrum exposed via `ModuleContext::cepstrum_buf`, gated by
`ModuleSpec::needs_cepstrum`. Only the Harmony Lifter sub-effect needs it in v1; the
infrastructure must impose zero cost when no slot declares the dependency.

**Architecture:** New `src/dsp/cepstrum.rs` owns a `CepstrumBuf` struct that wraps a
`realfft::RealFftPlanner`, scratch slices, and the output cepstrum. `Pipeline` holds one
`CepstrumBuf` per channel. Each block, Pipeline computes `any_needs_cepstrum` from active
slot specs. Inside the hop closure (after the main STFT-forward but before module dispatch),
when needed, Pipeline calls `cepstrum.compute_from_bins(&bins[..num_bins])`. The result is
exposed as `ctx.cepstrum_buf = Some(&cepstrum_buf[ch].quefrency()[..fft_size])`.

**Math (per `15-harmony.md` § a):**

```
log_mag[k] = 0.5 * ln(|bins[k]|² + ε)        ε = 1e-10
cepstrum[n] = inverse_real_fft(log_mag)      length = fft_size (real)
```

Cepstrum index 0 = DC of the log-spectrum (overall loudness), low n = formant envelope,
high n = pitch / harmonics. Per research findings (`research/02-pitch-and-cepstral.md`):
ε = 1e-10 on magnitude-squared; silence cross-fade at -60 dBFS RMS default.

**Tech stack:** `realfft` (already in deps), Phase 1's `cepstrum_buf: Option<&'block [f32]>`
field on `ModuleContext`, `assert_process_allocs` discipline (no per-block allocation).

// tests/cepstrum.rs

use spectral_forge::dsp::cepstrum::CepstrumBuf;
use num_complex::Complex;

#[test]
fn cepstrum_of_pure_tone_has_low_quefrency_envelope_and_pitch_spike() {
    let fft_size = 2048;
    let num_bins = fft_size / 2 + 1;
    let mut cb = CepstrumBuf::new(fft_size);

    // Build a complex bin vector representing a pure tone at bin 100.
    let mut bins = vec![Complex::<f32>::new(0.0, 0.0); num_bins];
    bins[100] = Complex::new(1.0, 0.0);
    bins[101] = Complex::new(0.5, 0.0);  // small leakage so log_mag isn't -inf elsewhere
    bins[99]  = Complex::new(0.5, 0.0);

    cb.compute_from_bins(&bins);

    let q = cb.quefrency();
    assert_eq!(q.len(), fft_size);
    // DC quefrency reflects overall log-energy; should be a finite, non-zero value.
    assert!(q[0].is_finite());
    // Cepstrum should be finite everywhere (epsilon clamp working).
    for (n, v) in q.iter().enumerate() {
        assert!(v.is_finite(), "quefrency[{}] = {}", n, v);
    }
}

#[test]
fn cepstrum_silent_input_produces_finite_zero_envelope() {
    let fft_size = 2048;
    let num_bins = fft_size / 2 + 1;
    let mut cb = CepstrumBuf::new(fft_size);

    let bins = vec![Complex::<f32>::new(0.0, 0.0); num_bins];
    cb.compute_from_bins(&bins);

    // Silence => log(epsilon) for every bin => cepstrum is just 0.5 * ln(eps) at DC and ~0 elsewhere.
    // Critical assertion: no NaN, no Inf.
    for (n, v) in cb.quefrency().iter().enumerate() {
        assert!(v.is_finite(), "silent quefrency[{}] = {}", n, v);
    }
}

fn cepstrum_buf_reusable_across_calls_no_alloc() {
    let fft_size = 2048;
    let num_bins = fft_size / 2 + 1;
    let mut cb = CepstrumBuf::new(fft_size);

    let bins = vec![Complex::<f32>::new(1.0, 0.0); num_bins];
    for _ in 0..10 {
        cb.compute_from_bins(&bins);
        assert!(cb.quefrency()[0].is_finite());
    }
}

```rust
// src/dsp/cepstrum.rs

use num_complex::Complex;
use realfft::{RealFftPlanner, RealToComplex, ComplexToReal};
use std::sync::Arc;

/// Magnitude-squared epsilon for log clamp. ε = 1e-10 ⇒ -100 dB log threshold.
/// Per `research/02-pitch-and-cepstral.md` Topic B Recommendation 5.
const LOG_EPSILON_SQ: f32 = 1e-10;

/// Per-channel cepstrum scratch + output. One instance per channel in Pipeline.
/// All buffers sized for the maximum supported FFT (`MAX_FFT_SIZE`).
pub struct CepstrumBuf {
    fft_size:   usize,
    /// Forward real-FFT planner used for log-mag → cepstrum (we *inverse*-transform the
    /// log-mag spectrum; `realfft` exposes this as `ComplexToReal`).
    inv_fft:    Arc<dyn ComplexToReal<f32>>,
    /// log_mag[k] = 0.5 * ln(|bins[k]|² + ε), scratch for inverse FFT input.
    /// Stored as Complex with imag=0 because `ComplexToReal` expects complex input.
    log_mag:    Vec<Complex<f32>>,
    /// Quefrency-domain output, length = fft_size.
    cepstrum:   Vec<f32>,
    /// realfft scratch (its API requires a separate scratch buffer).
    scratch:    Vec<Complex<f32>>,
}

impl CepstrumBuf {
    /// Allocate buffers for the given FFT size. May allocate; not RT-safe to call this on the audio thread.
    pub fn new(fft_size: usize) -> Self {
        let mut planner = RealFftPlanner::<f32>::new();
        let inv_fft     = planner.plan_fft_inverse(fft_size);
        let num_bins    = fft_size / 2 + 1;
        let scratch_len = inv_fft.get_scratch_len();
        Self {
            fft_size,
            inv_fft,
            log_mag:  vec![Complex::new(0.0, 0.0); num_bins],
            cepstrum: vec![0.0; fft_size],
            scratch:  vec![Complex::new(0.0, 0.0); scratch_len],
        }
    }

    /// Resize all internal buffers for a new FFT size. Only call from outside the audio thread
    /// (e.g. on `Pipeline::reset`). Caller must guarantee `fft_size` is a power of two ≤ MAX_FFT_SIZE.
    pub fn resize(&mut self, fft_size: usize) {
        if fft_size == self.fft_size { return; }
        *self = Self::new(fft_size);
    }

    /// Compute cepstrum from a slice of complex FFT bins. Reads `bins[..num_bins]`; writes into
    /// `self.cepstrum[..fft_size]`. RT-safe. Cost: 1 sqrt-free magnitude pass + 1 ln + 1 inverse FFT.
    ///
    /// Caller invariant: `bins.len() >= self.fft_size / 2 + 1`.
    pub fn compute_from_bins(&mut self, bins: &[Complex<f32>]) {
        let num_bins = self.fft_size / 2 + 1;
        debug_assert!(bins.len() >= num_bins);
        for k in 0..num_bins {
            let mag_sq = bins[k].norm_sqr();
            // ε clamp avoids -inf at silent bins.
            let log_v = 0.5 * (mag_sq + LOG_EPSILON_SQ).ln();
            self.log_mag[k] = Complex::new(log_v, 0.0);
        }
        // Inverse real-FFT: log_mag (Hermitian) → cepstrum (real, length fft_size).
        // `process_with_scratch` overwrites log_mag — that's fine, we rebuild it next call.
        let _ = self.inv_fft.process_with_scratch(
            &mut self.log_mag,
            &mut self.cepstrum,
            &mut self.scratch,
        );
    }

    /// Borrow the cepstrum output. Length = `fft_size`.
    #[inline]
    pub fn quefrency(&self) -> &[f32] {
        &self.cepstrum
    }
}
```
