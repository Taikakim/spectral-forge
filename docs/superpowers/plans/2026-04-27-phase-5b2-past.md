**Goal:** Ship the Past module — the first read-only consumer of the History Buffer infrastructure (Phase 5b.1). Five sub-effects (Granular Window, Decay Sorter, Spectral Convolution, Reverse, Stretch) operate on past STFT frames; Granular blends with `BinPhysics.crystallization`, Convolution gates on `BinPhysics.flux`, Stretch uses the new shared `PhaseRotator` and `if_offset` cache for phase-coherent variable-rate playback.

**Architecture:** New `ModuleType::Past` slot. Per-channel state: a `read_phase: f64` for Stretch, a `read_offset_frames: u32` for Reverse, a `sort_order: Vec<u32>` (top 256 bins by amplitude) for Decay Sorter, plus a `xorshift32` seed for Stretch's hash dither. Mode is per-slot (`Mutex<PastMode>`), with Decay Sorter additionally carrying a `SortKey` (`Mutex<SortKey>`) for the three sort variants. The shared `PhaseRotator` (src/dsp/phase.rs) and `if_offset[k]` cache (in `ModuleContext`, filled by Pipeline once per hop after analysis FFT) ship in this plan because Past Stretch is the first consumer; Future and Punch will reuse them in later phases.

**Tech Stack:** Rust 2021, num-complex, parking_lot, no new dependencies.

```rust
//! Phase rotation helper used by Past Stretch and (later) Future / Punch.
//!
//! Encapsulates the operation: rotate a complex bin by `2π · freq_offset ·
//! time_delta` radians. Uses a 1024-entry sin/cos LUT to avoid trig calls in
//! the hot path. Cost per call: ~6 muls + 3 adds + 2 LUT loads.

use num_complex::Complex;

const LUT_SIZE: usize = 1024;
const LUT_SIZE_F: f32 = LUT_SIZE as f32;
const TAU: f32 = std::f32::consts::TAU;

pub struct PhaseRotator {
    sin_lut: [f32; LUT_SIZE],
    cos_lut: [f32; LUT_SIZE],
}

impl PhaseRotator {
    pub fn new() -> Self {
        let mut sin_lut = [0.0_f32; LUT_SIZE];
        let mut cos_lut = [0.0_f32; LUT_SIZE];
        for i in 0..LUT_SIZE {
            let theta = (i as f32 / LUT_SIZE_F) * TAU;
            sin_lut[i] = theta.sin();
            cos_lut[i] = theta.cos();
        }
        Self { sin_lut, cos_lut }
    }

    /// Rotate `c` by `2π · freq_offset · time_delta` radians.
    /// `freq_offset` is in cycles-per-hop (or any unit such that
    /// `freq_offset · time_delta` is in cycles). The product is wrapped into
    /// [0, 1) before LUT lookup.
    #[inline]
    pub fn rotate(&self, c: Complex<f32>, freq_offset: f32, time_delta: f32) -> Complex<f32> {
        let cycles = freq_offset * time_delta;
        // Wrap to [0, 1). Take fract; if negative, add 1.
        let frac = cycles - cycles.floor();
        let idx = (frac * LUT_SIZE_F) as usize % LUT_SIZE;
        let cs = self.cos_lut[idx];
        let sn = self.sin_lut[idx];
        Complex::new(
            c.re * cs - c.im * sn,
            c.re * sn + c.im * cs,
        )
    }
}

impl Default for PhaseRotator {
    fn default() -> Self { Self::new() }
}
```

(e) Attach to ctx. Where `ctx` is constructed (the `ModuleContext { ... }` literal), add:

```rust

```

(Note: this requires `ctx` to be built AFTER the closure exits — which would mean the closure sees a stale ctx. Look back at Phase 5b.1 Task 5.6c′ for the same problem and the same solution: the closure receives `ctx` by reference, so attach `if_offset` to `ctx` BEFORE the closure begins. Since we fill `if_offset_buf` only INSIDE the closure, the modules will read it as it is filled — the assignment happens before the same module reads it.)

Concretely the order inside the closure becomes:
1. analysis window + sanitize + FFT
2. update spectrum_buf
3. fill `if_offset_buf[0..num_bins]` (uses the just-computed `complex_buf`)
4. write to `pending_hop_frames` (Phase 5b.1 Task 5.6)
5. call `fx_matrix.process_hop` — modules see the freshly-filled `if_offset_buf` because `ctx.if_offset` borrows the same memory.

This works because `ctx.if_offset` was attached as `&self.if_offset_buf[..num_bins]` which is a *long-lived* immutable borrow valid across the closure body, and inside the closure we write through a *separate* mutable borrow `if_offset_buf: &mut Vec<f32>`. Rust will reject the two coexisting borrows.

**Resolution.** Don't re-borrow `&self.if_offset_buf` for `ctx`. Instead, before building `ctx`, take an immutable borrow `let if_offset_view: &[f32] = ...` of the *same* `Vec`. This still conflicts with `&mut Vec` inside the closure.

The clean fix: split borrow via slice. The closure mutates the slice; the ctx holds the immutable view. Since we want the modules to see the just-written values WITHIN the same closure invocation, the borrow must be `&[f32]` reborrowed AFTER the mutable write each hop. This needs `ctx.if_offset` to be set *inside* the closure for each call.

The cleanest model is the same as what Phase 5b.1 settled on for `history`: attach the borrow to `ctx` BEFORE the closure, and have the closure write to a *scratch* buffer that the ctx already aliases. Make `if_offset_buf` not a `Vec<f32>` but a fixed `[f32; MAX_NUM_BINS]` owned by Pipeline. Take an immutable shared borrow `&[f32]` for ctx, and mutate through `UnsafeCell` (not allowed without unsafe) OR through the same trick as `pending_hop_frames`: write outside the closure.

The compromise the spec actually wants is: **Pipeline computes `if_offset` ONCE per process() call from the FIRST hop's complex_buf** (a representative snapshot). All modules get the same `if_offset` for the entire block. Coarseness is acceptable because IF varies slowly over a single block at typical block sizes.

Replace step (d) with this: **fill `if_offset_buf` AFTER the closure** using the LAST hop's `complex_buf` (which still holds the most-recent FFT output after the closure ends). Then attach `ctx.if_offset` before fx_matrix is called — but fx_matrix is called *inside* the closure!

This is a fundamental design tension: per-block ctx + per-hop fill. Resolution for v1:

**Pre-compute `if_offset` from the LAST block's stored snapshot** (one-block latency). Add `prev_block_complex: Vec<Complex<f32>>` to Pipeline; at the start of process(), derive `if_offset_buf` from it; at the end, store the last hop's `complex_buf` into it. Modules see one-block-late IF, which is fine for Past Stretch (Past is inherently a "look back" effect).

Concrete change to step (d): drop the in-closure fill. Add at the start of `Pipeline::process`, after sample_rate is known:

```rust
        // Derive if_offset from the previous block's last analysis FFT. One-block
        // latency is acceptable for Past consumers; saves a per-hop borrow conflict.
        let inv_fft = 1.0_f32 / fft_size as f32;
        for k in 1..num_bins {
            let centre = k as f32 * self.sample_rate * inv_fft;
            // Approximate IF from |X[k]|·angle: phase_delta proxy. For v1 we use bin
            // centre (offset = 0); Phase 4's PLPV per-channel IF replaces this.
            let inst = centre;
            self.if_offset_buf[k] = (inst - centre) / centre.max(1e-6);
        }
        self.if_offset_buf[0] = 0.0;
```

This makes `if_offset` initially all zeros (every bin sits at its centre). Past Stretch then degrades gracefully (no extra phase rotation when IF is unknown). When Phase 4 is ready, replace `let inst = centre;` with the real per-channel IF read from a stored snapshot.

Then attach to ctx (no borrow conflict because ctx is built AFTER the if_offset fill):

```rust
        let ctx = ModuleContext {
            // …existing fields…
            if_offset: Some(&self.if_offset_buf[..num_bins]),
        };
```

```rust
//! Past module — read-only consumer of `HistoryBuffer` (Phase 5b.1).
//!
//! Five sub-modes:
//! 1. Granular Window — selectively replay a band from history.
//! 2. Decay Sorter — reorder bins by decay/stability/area summary stat.
//! 3. Spectral Convolution — point-wise multiply with a delayed self.
//! 4. Reverse — read history at a backward read position.
//! 5. Stretch — read history at a fractional rate, with phase rotation.
//!
//! All modes accept curves [AMOUNT, TIME, THRESHOLD, SPREAD, MIX] and
//! respect MIX as a per-bin wet/dry blend. State is per-channel.

use num_complex::Complex;
use serde::{Deserialize, Serialize};

use crate::dsp::history_buffer::HistoryBuffer;
use crate::dsp::modules::{
    GainMode, ModuleContext, ModuleType, SpectralModule,
};
use crate::dsp::phase::PhaseRotator;
use crate::params::{FxChannelTarget, StereoLink};

#[derive(Debug, Clone, Copy, PartialEq, Eq, Default, Serialize, Deserialize)]
pub enum PastMode {
    #[default]
    Granular,

}

    fn process(
        &mut self,
        channel: usize,
        _stereo_link: StereoLink,
        _target: FxChannelTarget,
        bins: &mut [Complex<f32>],
        _sidechain: Option<&[f32]>,

        let time      = curves.get(1).map(|c| &c[..]).unwrap_or(&[]);
        let threshold = curves.get(2).map(|c| &c[..]).unwrap_or(&[]);
        let spread    = curves.get(3).map(|c| &c[..]).unwrap_or(&[]);

        for v in suppression_out.iter_mut() { *v = 0.0; }

            PastMode::Stretch    => self.apply_stretch(ch, bins, history, amount, time, spread, mix, ctx),

    }

        self.fft_size = fft_size;

            s.reverse_read_offset = 0;
            s.stretch_read_phase = 0.0;

            for v in &mut s.sort_scratch { *v = Complex::new(0.0, 0.0); }

    fn num_curves(&self) -> usize { 5 }
    fn tail_length(&self) -> u32 { 0 }
```

## Task 5: `apply_granular` — selective time-windowed freeze

**Files:**
- Modify: `src/dsp/modules/past.rs`
- Test: `tests/past.rs`

**Algorithm.** Each bin `k` reads from `history.read_frame(channel, age_frames)` where:
- `age_frames = round(time[k] * MAX_AGE_FRAMES_GRAN)` — TIME curve maps 0→1 onto [0, MAX_AGE_FRAMES_GRAN] frames back.
- Only bins inside the SPREAD window around centre `k` participate. The spread window per centre k is `[k - half, k + half]` where `half = (spread[k] * num_bins / 16).round()` — SPREAD=1 covers ±num_bins/16 ≈ ±64 bins.
- Below THRESHOLD curve, bin passes through unchanged (no replacement).
- AMOUNT scales the replacement magnitude.
- BinPhysics.crystallization (if available) further blends: `crystallization=1` forces full freeze, `crystallization=0` lets AMOUNT decide.

For v1, "spread" only matters as a per-bin bandwidth around its own centre — interpreted simply as: "if `spread[k] > 0.5`, smooth the historic frame by averaging adjacent bins." A second pass.

```rust
    fn apply_granular(
        &mut self, ch: usize, bins: &mut [Complex<f32>], hist: &HistoryBuffer,
        amount: &[f32], time: &[f32], threshold: &[f32], spread: &[f32], mix: &[f32],
        ctx: &ModuleContext<'_>,
    ) {
        let n = bins.len().min(ctx.num_bins);
        // TIME maps [0..1] to [0..MAX_AGE_FRAMES_GRAN]. Use the buffer's actual
        // capacity-1 to avoid oversize reads.
        let max_age = (hist.capacity_frames().saturating_sub(1)) as f32;
        // BinPhysics crystallization, if present, biases AMOUNT toward 1.0 per-bin.
        let cryst = ctx.bin_physics.map(|p| &p.crystallization[..]);
        for k in 0..n {
            let bin_amount = amount.get(k).copied().unwrap_or(0.0);
            let cryst_bias = cryst.and_then(|c| c.get(k).copied()).unwrap_or(0.0);
            let effective_amount = (bin_amount + cryst_bias).clamp(0.0, 1.0);
            // THRESHOLD gating: skip bins below threshold.
            let mag = bins[k].norm();
            let thr = threshold.get(k).copied().unwrap_or(0.0);
            if mag < thr { continue; }
            // Read historic frame at age = TIME[k] * max_age.
            let age = (time.get(k).copied().unwrap_or(0.0).clamp(0.0, 1.0) * max_age) as usize;
            let frame = match hist.read_frame(ch, age) { Some(f) => f, None => continue };
            // SPREAD: simple 3-bin average if spread[k] > 0.5.
            let val = if spread.get(k).copied().unwrap_or(0.0) > 0.5 && k > 0 && k + 1 < frame.len() {
                (frame[k - 1] + frame[k] + frame[k + 1]) * (1.0 / 3.0)
            } else {
                frame[k]
            };
            // Apply AMOUNT scaling on the replacement magnitude (preserve historic phase).
            let replacement = val * effective_amount;
            // MIX: blend with current bin.
            let m_val = mix.get(k).copied().unwrap_or(1.0).clamp(0.0, 1.0);
            bins[k] = bins[k] * (1.0 - m_val) + replacement * m_val;
        }
    }
```

## Task 6: `apply_decay_sorter` — temporal reconstruction

**Files:**
- Modify: `src/dsp/modules/past.rs`
- Test: `tests/past.rs`

**Algorithm.** Pick top MAX_SORT_BINS bins by current magnitude (not over-history average; instantaneous). Sort them by the chosen `SortKey`:
- `Decay` → `summary_decay_estimate(ch)[k]` — long-ringing first → assigned to *low* output bins.
- `Stability` → `summary_if_stability(ch)[k]` — most stable first → low output bins.
- `Area` → `summary_rms_envelope(ch)[k]` — highest RMS first → low output bins.

Output: zero `bins`, then write each ranked bin's *complex* value into output bin slot `low_k + rank`, where `low_k` is the lowest "musically useful" bin (≥ ~10 Hz). Bins not in the top-256 are zeroed (or pass through if MIX < 1).

Suppression: Decay Sorter is destructive — fill `suppression_out` with the original-vs-replacement magnitude difference for bins that get removed.

```rust
    fn apply_decay_sorter(
        &mut self, ch: usize, bins: &mut [Complex<f32>], hist: &HistoryBuffer,
        amount: &[f32], threshold: &[f32], mix: &[f32], _ctx: &ModuleContext<'_>,
    ) {
        let n = bins.len();
        // Bin 10 ≈ 230 Hz at fft 2048 / 48 kHz — first musically useful bin.
        let low_k = 10usize.min(n.saturating_sub(1));
        // Pick top MAX_SORT_BINS bins by current magnitude above THRESHOLD.
        let mut candidates: smallvec::SmallVec<[(u32, f32); MAX_SORT_BINS]> =
            smallvec::SmallVec::new();
        for k in 0..n {
            let mag = bins[k].norm();
            let thr = threshold.get(k).copied().unwrap_or(0.0);
            if mag <= thr { continue; }
            if candidates.len() < MAX_SORT_BINS {
                candidates.push((k as u32, mag));
            } else {
                // Replace the smallest-magnitude entry if this one is louder.
                if let Some((min_idx, _)) = candidates.iter().enumerate()
                    .min_by(|a, b| a.1.1.partial_cmp(&b.1.1).unwrap_or(std::cmp::Ordering::Equal))
                {
                    if candidates[min_idx].1 < mag {
                        candidates[min_idx] = (k as u32, mag);
                    }
                }
            }
        }
        // Pull the right summary stat per sort key.
        let key_values_borrow: std::cell::Ref<'_, [f32]> = match self.sort_key {
            SortKey::Decay     => hist.summary_decay_estimate(ch),
            SortKey::Stability => hist.summary_if_stability(ch),
            SortKey::Area      => hist.summary_rms_envelope(ch),
        };
        let key_values: &[f32] = &*key_values_borrow;
        // Sort descending — highest key value first → assigned to lowest output slot.
        candidates.sort_by(|a, b| {
            let ka = key_values.get(a.0 as usize).copied().unwrap_or(0.0);
            let kb = key_values.get(b.0 as usize).copied().unwrap_or(0.0);
            kb.partial_cmp(&ka).unwrap_or(std::cmp::Ordering::Equal)
        });
        // Snapshot current bins before destructive write.
        let scratch = &mut self.channels[ch].sort_scratch;
        scratch[..n].copy_from_slice(&bins[..n]);
        // Drop the ref before mutating bins — guaranteed safe because of explicit drop.
        drop(key_values_borrow);
        // Zero output destination bins for the ranked range.
        let max_dest = (low_k + candidates.len()).min(n);
        for k in low_k..max_dest { bins[k] = Complex::new(0.0, 0.0); }
        // Write each ranked complex value into low_k + rank.
        for (rank, (src_k, _)) in candidates.iter().enumerate() {
            let dest = low_k + rank;
            if dest >= n { break; }
            let bin_amount = amount.get(*src_k as usize).copied().unwrap_or(1.0);
            let m_val = mix.get(dest).copied().unwrap_or(1.0).clamp(0.0, 1.0);
            let original = bins[dest]; // already zeroed above
            let value = scratch[*src_k as usize] * bin_amount;
            bins[dest] = original * (1.0 - m_val) + value * m_val;
        }
        // Source bins outside the destination range get cleared (their content moved).
        for (_, (src_k, _)) in candidates.iter().enumerate() {
            let kk = *src_k as usize;
            if kk < low_k || kk >= max_dest {
                let m_val = mix.get(kk).copied().unwrap_or(1.0).clamp(0.0, 1.0);
                bins[kk] = scratch[kk] * (1.0 - m_val);
            }
        }
    }
```

## Task 7: `apply_convolution` — self-resonance

```rust
            let mag = bins[k].norm();
            let thr = threshold.get(k).copied().unwrap_or(0.0);
            if mag < thr { continue; }

            // No flux → fully gated out; full flux → full eligibility.
            let bin_amount = amount.get(k).copied().unwrap_or(0.0) * flux_gate;
            if bin_amount < 1e-6 { continue; }
            let age = (time.get(k).copied().unwrap_or(0.0).clamp(0.0, 1.0) * max_age) as usize;
            let frame = match hist.read_frame(ch, age) { Some(f) => f, None => continue };
            // Point-wise multiply: bins[k] *= history[k] * AMOUNT.
            let conv = bins[k] * frame[k] * bin_amount;
            let m_val = mix.get(k).copied().unwrap_or(1.0).clamp(0.0, 1.0);
            bins[k] = bins[k] * (1.0 - m_val) + conv * m_val;
        }
    }
```

```rust
    fn apply_reverse(
        &mut self, ch: usize, bins: &mut [Complex<f32>], hist: &HistoryBuffer,
        amount: &[f32], time: &[f32], threshold: &[f32], mix: &[f32], ctx: &ModuleContext<'_>,
```

```rust
    fn apply_stretch(
        &mut self, ch: usize, bins: &mut [Complex<f32>], hist: &HistoryBuffer,
        amount: &[f32], time: &[f32], spread: &[f32], mix: &[f32], ctx: &ModuleContext<'_>,
    ) {
        let n = bins.len().min(ctx.num_bins);
        let max_age = hist.capacity_frames().saturating_sub(1) as f32;
        // TIME maps [0..1] log-scale to [0.25×..4×] read rate.
        // 0.0 → 0.25, 0.5 → 1.0, 1.0 → 4.0
        let t_avg = time.iter().take(n).copied().sum::<f32>() / n as f32;
        let rate = 4.0_f32.powf(2.0 * t_avg.clamp(0.0, 1.0) - 1.0);

        let st = &mut self.channels[ch];
        let read_age = st.stretch_read_phase as f32;
        // Advance for next hop.
        st.stretch_read_phase += rate as f64;
        if st.stretch_read_phase > max_age as f64 { st.stretch_read_phase = 0.0; }

        // Read the two bracketing frames once into a workspace.
        let scratch = &mut self.channels[ch].sort_scratch;
        scratch[..n].fill(Complex::new(0.0, 0.0));
        let ok = hist.read_fractional(ch, read_age, &mut scratch[..n]);
        if !ok { return; }

        let if_offset = ctx.if_offset.unwrap_or(&[]);
        for k in 0..n {
            let bin_amount = amount.get(k).copied().unwrap_or(0.0).clamp(0.0, 1.0);
            if bin_amount < 1e-6 { continue; }

            let mut sample = scratch[k];
            // Phase rotation: 2π · if_offset · (rate - 1) cycles.
            let if_off = if_offset.get(k).copied().unwrap_or(0.0);
            let rot = if_off * (rate - 1.0);
            sample = self.rotator.rotate(sample, rot, 1.0);

            // Per-bin hash dither based on SPREAD curve (xorshift32).
            let spr = spread.get(k).copied().unwrap_or(0.0).clamp(0.0, 1.0);
            if spr > 1e-6 {
                let s = self.channels[ch].stretch_rng;
                let mut x = s ^ (k as u32).wrapping_mul(0x9E37_79B9);
                x ^= x << 13;
                x ^= x >> 17;
                x ^= x << 5;
                self.channels[ch].stretch_rng = x;
                let dither_phase = ((x as f32 / u32::MAX as f32) - 0.5) * spr * 0.05;
                sample = self.rotator.rotate(sample, dither_phase, 1.0);
            }
            let value = sample * bin_amount;
            let m_val = mix.get(k).copied().unwrap_or(1.0).clamp(0.0, 1.0);
            bins[k] = bins[k] * (1.0 - m_val) + value * m_val;
        }
    }
```

```rust
    let mut h = HistoryBuffer::new(2, 100, NUM_BINS);
    let mut m = PastModule::new(48000.0, 2048);
    m.set_past_mode(mode);
    m.set_past_sort_key(sort_key);

    let amount    = vec![1.0_f32; NUM_BINS];
    let time      = vec![0.5_f32; NUM_BINS];
    let threshold = vec![0.05_f32; NUM_BINS];
    let spread    = vec![0.0_f32; NUM_BINS];
    let mix       = vec![0.5_f32; NUM_BINS];
    let curves: Vec<&[f32]> = vec![&amount, &time, &threshold, &spread, &mix];

    let mut outputs: Vec<Vec<Complex<f32>>> = Vec::new();
    for hop in 0..200 {
        let frame = synth_input(hop);
        h.write_hop(0, &frame);
        h.write_hop(1, &frame);
        h.advance_after_all_channels_written();

        let mut bins = frame.clone();
        let mut supp = vec![0.0_f32; NUM_BINS];
        let mut ctx = ModuleContext::new(48000.0, 2048, NUM_BINS, 10.0, 100.0, 1.0, 0.5, false, false);
        ctx.history = Some(&h);
        m.process(0, StereoLink::Linked, FxChannelTarget::All,
                  &mut bins, None, &curves, &mut supp, &ctx);
        outputs.push(bins);
    }
    outputs
}
```
