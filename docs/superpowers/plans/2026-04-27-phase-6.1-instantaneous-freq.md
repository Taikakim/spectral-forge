> **STATUS:** IMPLEMENTED — landed 2026-04-30 across commits `6f37c4a`..`b2e06b2`
> on branch `feature/next-gen-modules-plans`. Code is the source of truth.
> This plan is kept for history only. Authoritative status: `docs/superpowers/STATUS.md`.

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development`
> (recommended) or `superpowers:executing-plans` to implement this plan task-by-task.
> Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Populate `ModuleContext::instantaneous_freq` per-hop in `Pipeline`, computed
once and shared with all modules that declare `needs_instantaneous_freq`.

**Architecture:** A new helper `src/dsp/instantaneous_freq.rs` exposes a SIMD-friendly
scalar kernel `compute_instantaneous_freq()`. `Pipeline` pre-allocates per-channel
`if_buffer` and `prev_phase` arrays. At the start of each block, Pipeline scans active
slot specs to determine `any_needs_if`; if true, after each STFT-forward call inside the
hop closure, Pipeline computes IF and exposes it via `ctx.instantaneous_freq =
Some(&if_buffer[ch][..num_bins])`. Otherwise the field stays `None` and the cost is zero.

**Tech stack:** Rust, `num_complex::Complex`, existing `Pipeline::process()` STFT loop,
Phase 1's `ModuleSpec.needs_instantaneous_freq: bool`, Phase 4's `ctx.unwrapped_phase`
(used opportunistically when present; falls back to principal-argument unwrap when None).

**Cross-phase notes:** Assumes Phase 1 has added `instantaneous_freq: Option<&'block [f32]>`
to `ModuleContext` (currently `None` at all callsites). This plan populates the field.

use std::f32::consts::PI;

/// Wrap a phase value into (-π, π].
/// Used by IF computation to map phase deviation back into the principal range.
#[inline(always)]
pub fn principal_argument(x: f32) -> f32 {
    // Two-step wrap; faster than `rem_euclid` for the typical small-deviation case.
    let mut x = x;
    while x > PI { x -= 2.0 * PI; }
    while x <= -PI { x += 2.0 * PI; }
    x
}

// src/dsp/instantaneous_freq.rs (append)

/// Compute per-bin instantaneous frequency in Hz from two consecutive frames of
/// wrapped phase, using the standard phase-vocoder formula:
///
///   IF[k] = bin_center_freq[k]
///         + principal_argument(curr[k] - prev[k] - expected_phase_inc[k]) × sr / (hop × 2π)
///
/// where `expected_phase_inc[k] = 2π × k × hop_size / fft_size`.
///
/// `if_out`, `prev`, `curr` must all be at least `num_bins = fft_size / 2 + 1` long;
/// only the first `num_bins` entries are touched. DC bin (k=0) is always set to 0.0.
///
/// No allocation. ~5 FLOPs per bin (1 sub, 1 add, 1 div, 1 fma, 1 wrap).
#[inline]
pub fn compute_instantaneous_freq(
    prev:        &[f32],
    curr:        &[f32],
    if_out:      &mut [f32],
    sample_rate: f32,
    hop_size:    usize,
    fft_size:    usize,
) {
    debug_assert!(prev.len() >= fft_size / 2 + 1);
    debug_assert!(curr.len() >= fft_size / 2 + 1);
    debug_assert!(if_out.len() >= fft_size / 2 + 1);
    let num_bins = fft_size / 2 + 1;
    let bin_freq_hz       = sample_rate / fft_size as f32;
    let hop_per_fft       = hop_size as f32 / fft_size as f32;
    let radians_to_hz_div = sample_rate / (hop_size as f32 * std::f32::consts::TAU);

    if_out[0] = 0.0;
    for k in 1..num_bins {
        let expected_phase_inc = std::f32::consts::TAU * k as f32 * hop_per_fft;
        let raw_delta          = curr[k] - prev[k] - expected_phase_inc;
        let dev                = principal_argument(raw_delta);
        if_out[k] = (k as f32) * bin_freq_hz + dev * radians_to_hz_div;
    }
}
```

            // Skip when no active module needs it — saves ~1 FLOP/bin/hop/ch.
            if any_needs_if {
                let if_out      = &mut if_buffer[ch_idx];
                let prev_phase  = &mut prev_phase[ch_idx];
                // Read current wrapped phase into a scratch slice.
                // We reuse `if_out` as scratch then overwrite it; that is fine because
                // compute_instantaneous_freq reads `prev` and `curr` and writes `if_out`.
                let mut curr_phase_scratch = [0.0_f32; crate::dsp::pipeline::MAX_NUM_BINS];
                for k in 0..num_bins {
                    curr_phase_scratch[k] = bins[k].arg();
                }
                crate::dsp::instantaneous_freq::compute_instantaneous_freq(
                    &prev_phase[..num_bins],
                    &curr_phase_scratch[..num_bins],
                    &mut if_out[..num_bins],
                    sample_rate,
                    hop_size,
                    fft_size,
                );
                prev_phase[..num_bins].copy_from_slice(&curr_phase_scratch[..num_bins]);
            }
```

```rust
// tests/instantaneous_freq.rs (append)

#[test]
fn if_kernel_independent_per_channel_invocation() {
    // Two simulated channels with different inputs. The kernel itself is stateless
    // per call; this test pins the per-call separation.
    let sample_rate = 48000.0;
    let fft_size    = 2048;
    let hop_size    = 512;
    let num_bins    = fft_size / 2 + 1;

    let mut prev_l = vec![0.0_f32; num_bins];
    let mut prev_r = vec![0.0_f32; num_bins];
    let mut curr_l = vec![0.0_f32; num_bins];
    let mut curr_r = vec![0.0_f32; num_bins];
    // L: 1 kHz; R: 2 kHz.
    let bin_freq_hz = sample_rate / fft_size as f32;
    let bin_l = (1000.0_f32 / bin_freq_hz).round() as usize;
    let bin_r = (2000.0_f32 / bin_freq_hz).round() as usize;
    let frame_seconds = (hop_size as f32) / sample_rate;
    curr_l[bin_l] = 2.0 * std::f32::consts::PI * 1000.0 * frame_seconds;
    curr_r[bin_r] = 2.0 * std::f32::consts::PI * 2000.0 * frame_seconds;

    let mut if_l = vec![0.0_f32; num_bins];
    let mut if_r = vec![0.0_f32; num_bins];
    compute_instantaneous_freq(&prev_l, &curr_l, &mut if_l, sample_rate, hop_size, fft_size);
    compute_instantaneous_freq(&prev_r, &curr_r, &mut if_r, sample_rate, hop_size, fft_size);

    assert!((if_l[bin_l] - 1000.0).abs() < 1.0);
    assert!((if_r[bin_r] - 2000.0).abs() < 1.0);
    // L should not see 2 kHz at bin_r and vice versa (we wrote zeros there).
    // For zero-prev / zero-curr at a bin, IF reduces to bin_center_freq[k] (deviation = 0).
    assert!((if_l[bin_r] - bin_r as f32 * bin_freq_hz).abs() < 1e-3);
    assert!((if_r[bin_l] - bin_l as f32 * bin_freq_hz).abs() < 1e-3);
}
```
